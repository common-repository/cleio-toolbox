Cleio ToolBox is packed with nifty widgets and shortcodes that will cover all the basics of blogging... and more!

== Description ==

= Key Features =

* New Post Type and Widget: Photo Post, to feature your latest photo posts
* Pretty Archives: Shortcode generator to produce a filterable archive of all your content, with light AJAX tab navigation
* Biography Widget: Short biography, avatar (choose between uploading one, or using your WordPress profile one) and read more link
* RSS filter: exclude or add specific content or content types in your RSS feed
* Loop filter: exclude or add specific content or content types in your home/blog loop
* Social Networks Manager: register all your data once and for all, and display it easily with our widget
* Simple Instagram Widget: displays a feed of your latest pictures
* Simple Twitter Widget: customizes and displays a feed of your latest tweets
* Facebook Like Box Widget: customizes and displays a Facebook Like Box
* Elegant, light and clutter-free framework to manage all these options

== Installation ==

1. Browse to the Plugins Menu in WordPress. Then either search for « Cleio ToolBox » through the Search Tab, OR upload the « cleio-toolbox-zip » file through the Uploads tab.
2. Activate the plugin by going back to the Plugins Menu.
3. Setup and configure Cleio ToolBox by clicking the new « Cleio ToolBox » WordPress Menu.
4. If necessary, follow instructions in the framework, or in our online documentation http://www.cleio.co/documentation/cleio-toolbox/
